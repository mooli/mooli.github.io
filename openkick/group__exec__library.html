enum exec::Resident::Flags : uint8_t

resident module flags

Enumerator
RTF_COLDSTART 

module is initialised immediately after starting multitasking (V33)

RTF_SINGLETASK 

module is initialised while still in single-tasking mode (poorly documented: definitely not V33, probably V36+)

RTF_AFTERDOS 

module is initialised when DOS loads (V36+).

Note
should have priority -120 or lower.
RTF_AUTOINIT 

set if auto_init is the address of the initialisation function, otherwise it's a pointer to a MakeLibrary structure
